### Team Members

## Project Overview

PromptShield AI is an AI-powered web application that detects Prompt Injection attacks against Large Language Models (LLMs). It analyzes user prompts, classifies attack types, assigns a risk score, explains why the prompt is unsafe, and suggests safer alternatives.

## Planned Features

- Prompt Injection Detection
- Attack Classification
- Risk Score
- Explainability
- Safe Prompt Rewriting
- Detection Dashboard
